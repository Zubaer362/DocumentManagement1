# Document Management App

A full-stack document management application for creating and managing quotations, invoices, and receipts.

## Features

- **Quotation Management**: Create professional quotations with client details, services, and pricing
- **Invoice Generation**: Convert quotations to invoices with automatic ID generation
- **Receipt Creation**: Generate payment receipts with multiple payment methods
- **PDF Export**: Download documents as professional PDF files
- **Email Integration**: Send documents via email (configurable)
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

### Backend
- Node.js & Express.js
- MongoDB & Mongoose
- PDFKit for PDF generation
- Nodemailer for email functionality

### Frontend
- React.js with functional components
- React Router for navigation
- Tailwind CSS for styling
- Fetch API for HTTP requests
